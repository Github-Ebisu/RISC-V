# RISC-V pipeline 
Design RISC-V 5 stages for some instructions:
- R-type :
  + add
  + sub
  + sll
  + slt
  + sltu
  + xor
  + srl
  + sra
  + or
  + and
- I-type :
  + addi
  + slti
  + sltiu
  + xori
  + ori
  + andi
  + slli
  + srli
  + srai
- B-type :
  + beq
  + bne
  + blt
  + bge
  + bltu
  + bgeu
- J-type :
  + jal
  + jalr
- LW, SW
- Lui, Auipc
